## Environment variables:

| Name               | Required | Default  | Description                     |
|:-------------------|:--------:|:---------|:--------------------------------|
| MONGO_DIAL_TIMEOUT | -        | 10       | MongoBD dial timeout in seconds |
| MONGO_DSN          | true     | -        | MongoBD DSN connection string   |
